# Mall-Customer-Segmentation-using-K-Means-Clustering

## Overview

This project applies **K-Means Clustering** to segment mall customers based on their purchasing behavior. Customer segmentation helps businesses identify groups of customers with similar characteristics, allowing them to create targeted marketing strategies and improve customer satisfaction.

## Objective

The main goal of this project is to:

* Explore customer data.
* Preprocess and scale numerical features.
* Determine the optimal number of clusters using the Elbow Method.
* Apply the K-Means clustering algorithm.
* Visualize customer segments using 2D and 3D plots.
* Interpret the resulting customer groups.

## Dataset

**Mall Customers Dataset**

### Features Used

* Age
* Annual Income (k$)
* Spending Score (1–100)

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Plotly
* Scikit-learn

## Project Workflow

1. Load the dataset.
2. Explore and clean the data.
3. Select relevant features.
4. Scale the features using **StandardScaler**.
5. Use the **Elbow Method** to determine the optimal number of clusters.
6. Train the **K-Means** model.
7. Assign each customer to a cluster.
8. Visualize the clusters using:

   * 3D Interactive Scatter Plot (Plotly)
9. Analyze customer segments.

## Results

The K-Means algorithm successfully grouped customers into distinct clusters based on their age, annual income, and spending behavior. These segments can support business decisions such as personalized marketing campaigns, customer targeting, and loyalty program planning.
